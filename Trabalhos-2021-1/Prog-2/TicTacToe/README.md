# Objetivo do programa
Este programa implementa o jogo da velha.

## Funções auxiliares
- O jogo é organizado em matriz.
- O jogador escolhe a linha e a coluna para marcar.
- O jogador escolhe X ou O.
- Exibe o jogo após a escolha.
